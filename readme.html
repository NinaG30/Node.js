Installer NodeJS

1 - https://nodejs.org/en/ Télécharger la version LTS (dernière version stable)
2 - Vérifier la version qui s'est installée dans le terminal de powershell en tapant "node -v"
3 - Initialiser le projet avec npm init afin de créer un fichier package.json (npm est le gestionnaire de paquets par défaut qui s'installe en même temps que l'environnement js de bas niveau Node.js)
4 - Installer les packages qui nous interessent, en l'occurrence Express et nodemon avec "npm install nomdupackage" . Toutefois, comme Nodemon n'est utilisé que pour le développement, cela sera plutôt "npm install nodemon --save-dev" (Nodemon est un package qui met à jour notre serveur démarré à chaque changement de fichier, ce qui facilite le développement Node)
5 - Créér un fichier json avec nos datas
6 - Créer le fichier "main" (app.js dans ce cas-ci)
